# 🛵 taximetro-pro-passageiro

Aplicativo web (PWA) de taxímetro para uso próprio.

## 🚀 Funções
- Cálculo automático por KM
- Tarifas inteligentes (Econômica, Padrão e Premium)
- Mínimo automático (R$7 dia / R$8 noite)
- Integração com Google Maps
- Funciona offline (PWA)

## 📲 Como usar
1. Abra o app
2. Clique em INICIAR
3. Use o Google Maps para definir o destino
4. O valor será calculado automaticamente

## 📦 Instalação
- Abra o site no celular
- Clique em "Adicionar à tela inicial"
